# Numerical verification of Muon g-2 using Nash-4 regularized Mandelbrot attractor (UCP-E8++ : Electroweak Sector, Muon g − 2, Weak angle and Electric Charge from E8 Lattice Projection)

This repository contains the numerical verification code for the muon anomalous magnetic moment calculation presented in:

**"UCP-E8++ Paper II: Electroweak Sector, Muon g-2, and Electric Charge from E₈ Lattice Projection"**

by Sergio Garnelo Cortés, Jazmín Dessiret López Incio, Edmundo Márquez Quintos  
Opus 2G Group, SAPI de CV, Puebla, México

## Overview

This code verifies the theoretical prediction Δa_μ = (249 ± 5) × 10⁻¹¹ using the Nash-4 regularized Mandelbrot attractor, and compares it with the Fermilab experimental value (251 ± 59) × 10⁻¹¹.
